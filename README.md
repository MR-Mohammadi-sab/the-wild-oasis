## 🧠 About the Project

The Wild Oasis is a full-stack hotel management system where users can:

- Browse available cabins
- Make reservations
- View booking history
- Manage their profile

Admins can:

- Manage cabins
- Track reservations
- Control availability
- Handle bookings

---

## 🛠️ Tech Stack

- Next.js (App Router)
- React
- JavaScript
- Tailwind CSS
- Server Actions
- REST API
